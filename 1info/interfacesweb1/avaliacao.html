Desenvolvimento de Interface Visual

Objetivo e Instruções

Objetivo: Construir a interface visual de um sistema de abertura de chamados, aplicando boas práticas de HTML semântico e estruturação CSS (Box Model, Flexbox e Efeitos Visuais).

Instruções Gerais:

  • Você deve criar dois arquivos: index.html (estrutura) e style.css (estilos visuais).
  • A consulta ao "Guia de Sintaxe CSS" e aos materiais das aulas é permitida e incentivada.

Requisitos do Projeto

1 Estrutura Global (HTML Semântico)

  • A página deve conter um cabeçalho (<header>), uma área de conteúdo principal (<main>) e um rodapé (<footer>).

  • O título da página (dentro do header) deve ser "Abertura de Chamado de TI".

2 O Formulário de Chamado (HTML)

Dentro do <main>, crie um formulário estruturado utilizando <fieldset> e <legend> (ex: "Detalhes do Incidente"). O formulário deve conter os seguintes campos, acompanhados de suas respectivas etiquetas (<label>):

Nome Completo

type="text"

E-mail Corporativo

type="email"

Setor

Crie um <select> com 3 opções de departamentos (ex: RH, Financeiro, Diretoria).

Nível de Urgência

Crie botões de rádio para Baixa, Média e Alta.

Descrição do Problema

Utilize uma <textarea>.

Botão de Envio

<button type="submit">

3 Estilização e Tráfego (CSS Avançado)

Fundo Global:

O body da página deve ter um plano de fundo utilizando linear-gradient (escolha duas cores profissionais).

O Card do Formulário (Box Model):

O formulário não deve ficar encostado nas bordas da tela. Ele deve estar dentro de uma caixa branca centralizada (margin: 0 auto;), com largura máxima (max-width) definida e padding adequado para dar "respiro" interno.

Flexbox:

Utilize display: flex e aplique flex-direction: column no formulário para empilhar os campos perfeitamente na vertical. Use a propriedade gap para criar o espaçamento automático entre eles.

Profundidade:

Aplique border-radius para arredondar os cantos do cartão do formulário e adicione um box-shadow suave para dar a impressão de que a caixa está "flutuando" sobre o fundo degradê.

Interação (Hover):

O botão de envio deve reagir ao mouse. Crie uma regra :hover mudando levemente a cor de fundo ou adicionando uma transição suave.
