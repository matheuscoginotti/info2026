O Manual do Construtor Visual

Este material serve como uma folha de consulta rápida (Cheat Sheet) para a sua atividade prática. Aqui você encontrará a sintaxe exata e as propriedades mais importantes mapeadas durante o nosso curso.

Lembre-se: No desenvolvimento profissional, você não precisa decorar cada linha de código, mas sim entender a lógica de aplicação e saber onde consultar a sintaxe correta quando necessário.

1. Seletores e Estrutura Básica

Regras básicas de como apontar para um elemento HTML e aplicar modificações:

/* SELETOR DE TIPO: Afeta todas as tags desse tipo na página */
body {
    background-color: #fafafa;
}

/* SELETOR DE CLASSE: Aplica-se a qualquer elemento que utilize o atributo class="..." */
.card-azul {
    background-color: lightblue;
}

/* SELETOR DE ID: Afeta apenas o elemento único com o atributo id="..." correspondente */
#botao-enviar {
    background-color: green;
}

/* PSEUDO-CLASSE :HOVER: Aplica regras visuais apenas quando o mouse passa sobre o elemento */
.card-azul:hover {
    background-color: deepskyblue;
}

2. Tipografia e Formatação de Texto

Controle total sobre fontes, tamanhos, pesos e o espaçamento de leitura:

.texto-customizado {
    font-family: 'Inter', Arial, sans-serif; /* Define a família da fonte e alternativas */
    font-size: 16px;                        /* Tamanho da fonte (px, em, rem) */
    font-weight: 700;                       /* Peso/Espessura: 300 (fina), 400 (normal), 700 (negrito) */
    color: #2c3e50;                         /* Cor do texto em formato hexadecimal */
    text-align: center;                     /* Alinhamento: left, right, center, justify */
    text-transform: uppercase;              /* Força caixa alta: uppercase, lowercase, none */
    line-height: 1.6;                       /* Espaçamento entre linhas (melhora a leitura) */
    letter-spacing: 1px;                    /* Espaçamento horizontal entre as letras */
}

3. O Modelo de Caixa (Box Model)

Configuração de dimensões, preenchimentos internos, margens e bordas físicas:

.caixa-estruturada {
    /* ATENÇÃO: Inclui padding e border dentro da largura total, evitando quebras */
    box-sizing: border-box; 

    width: 100%;                            /* Ocupa toda a largura disponível */
    max-width: 450px;                       /* Define um limite máximo fixo para a largura */
    height: auto;                           /* Altura adapta-se ao conteúdo interno automaticamente */

    /* PADDING: Espaço interno (entre o texto e a borda) */
    padding: 25px;                          /* Aplica 25px em todos os 4 lados */
    padding: 15px 30px;                     /* Atalho: 15px (topo/baixo) e 30px (lados) */

    /* MARGIN: Espaço externo (afastamento entre caixas vizinhas) */
    margin: 0 auto;                         /* Perfeito para centralizar blocos horizontais na tela */
    margin-bottom: 20px;                    /* Empurra elementos de baixo para longe */

    /* BORDER: Moldura física da caixa */
    border: 1px solid #cbd5e1;              /* Sintaxe: Espessura | Estilo (solid, dashed) | Cor */
}

4. Planos de Fundo e Transparências

Trabalhando com cores sólidas, gradientes dinâmicos e opacidade controlada:

.fundo-estilizado {
    background-color: #ffffff;              /* Fundo branco sólido tradicional */

    /* Fundo transparente via RGBA (Vermelho, Verde, Azul, Alfa de transparência) 
       Apenas o fundo fica transparente; os textos internos continuam 100% nítidos. */
    background-color: rgba(255, 255, 255, 0.85); 

    /* GRADIENTE LINEAR: Fusão de cores direcionada */
    background-image: linear-gradient(to right, #1e3a8a, #3b82f6); /* Roxo/Azul para a direita */
    background-image: linear-gradient(45deg, #0f172a, #1e293b);   /* Gradiente inclinado em 45 graus */

    /* OPACIDADE GLOBAL: Torna o bloco INTEIRO semi-transparente (fundo E texto juntos) */
    opacity: 0.6;                           /* Escala de 0.0 (invisível) a 1.0 (sólido) */
}

5. Flexbox (Alinhamento de Tráfego)

Gerenciamento inteligente de posicionamento e distribuição de elementos filhos:

.container-flexivel {
    display: flex;                          /* Habilita o modelo Flexbox no elemento pai */
    
    flex-direction: column;                 /* Empilha os itens verticalmente (ótimo para formulários) */
    flex-direction: row;                    /* Alinha os itens lado a lado em linha (padrão) */
    
    justify-content: center;                /* Alinhamento no eixo principal: flex-start, center, space-between */
    align-items: center;                    /* Alinhamento no eixo cruzado: flex-start, center, stretch */
    
    gap: 15px;                              /* Cria um espaçamento automático e idêntico entre os itens */
}

6. Bordas Suaves e Elevação Tridimensional

Tratamento de cantos arredondados, estouros de conteúdo e sombras projetadas:

.card-tridimensional {
    border-radius: 12px;                    /* Suaviza as 4 quinas retas da caixa */
    overflow: hidden;                       /* Corta cirurgicamente qualquer conteúdo filho que tente vazar */
    
    /* BOX-SHADOW (Sombra projetada): 
       Sintaxe: Deslocamento-X | Deslocamento-Y | Nível-de-Desfoque (Blur) | Cor-com-Opacidade */
    box-shadow: 0 10px 20px rgba(0, 0, 0, 0.15); 
    
    /* TRANSIÇÃO SUAVE: Amortece qualquer alteração de estilo (como no :hover) */
    transition: all 0.3s ease;              /* Altera o estado de forma suave durante 0.3 segundos */
}

/* Efeito combinado de elevação ao passar o mouse */
.card-tridimensional:hover {
    transform: translateY(-5px);            /* Desloca o elemento levemente para cima */
    box-shadow: 0 15px 30px rgba(0, 0, 0, 0.25); /* Deixa a sombra mais ampla e suave */
}
