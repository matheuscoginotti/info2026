Guia Rápido de Tags, Atributos e Estrutura Semântica

O Esqueleto Digital

Este guia rápido reúne as principais marcações (tags) estruturais do HTML5. Utilize-o para lembrar como abrir e fechar elementos corretamente durante a montagem das suas páginas e formulários.

1. A Estrutura Obrigatória

Todo documento HTML precisa desta base invisível para funcionar corretamente nos navegadores:

<!-- Declaração do HTML5 -->
<!DOCTYPE html>

<!-- Tag principal definindo o idioma -->
<html lang="pt-br">

<head>
    <!-- Suporte a acentuação e caracteres especiais -->
    <meta charset="UTF-8">
    <!-- Configuração essencial para telas de celular (Responsividade) -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Título da aba do navegador -->
    <title>Título da Minha Página</title>
    <!-- Vinculando o arquivo CSS externo -->
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <!-- Todo o conteúdo visível vai aqui dentro -->
</body>

</html>

2. Tags Semânticas (Organização)

Substitua o excesso de <div> por tags que dão significado real ao seu layout:

<header>
    <!-- Cabeçalho da página ou de uma seção -->
</header>

<main>
    <!-- Conteúdo principal e exclusivo da página (use apenas um por página) -->
</main>

<section>
    <!-- Uma seção genérica agrupando um tema específico -->
</section>

<footer>
    <!-- Rodapé contendo assinaturas, avisos legais ou links finais -->
</footer>

3. Textos, Links e Imagens

As marcações mais comuns para preencher o corpo do site com informações:

<!-- Títulos: h1 é o mais importante, h6 é o menor -->
<h1>Título Principal</h1>
<h2>Subtítulo Secundário</h2>

<!-- Parágrafo padrão -->
<p>Este é um bloco de texto normal com <strong>negrito</strong>.</p>

<!-- Imagem (Não precisa de tag de fechamento) -->
<img src="foto.jpg" alt="Descrição para acessibilidade">

<!-- Link (Âncora) -->
<a href="https://google.com" target="_blank">Abrir Google em nova aba</a>

4. Formulários (A Ponte de Interação)

A estrutura essencial para coletar dados do usuário de forma organizada:

<!-- A caixa do formulário -->
<form action="#" method="POST">

    <!-- Agrupamento visual e semântico (A borda com título) -->
    <fieldset>
        <legend>Dados de Contato</legend>

        <!-- Campo de Texto simples -->
        <!-- O 'for' do label DEVE ser igual ao 'id' do input -->
        <label for="nome">Nome Completo:</label>
        <input type="text" id="nome" name="nome_usuario" placeholder="João Silva" required>

        <!-- Menu Suspenso (Dropdown) -->
        <label for="setor">Departamento:</label>
        <select id="setor" name="setor">
            <option value="ti">Tecnologia</option>
            <option value="rh">Recursos Humanos</option>
        </select>

        <!-- Botões de Rádio (Apenas uma opção pode ser escolhida) -->
        <!-- O 'name' deve ser IGUAL em todas as opções do mesmo grupo -->
        <p>Nível de Urgência:</p>
        <input type="radio" id="urg-baixa" name="urgencia" value="baixa" checked>
        <label for="urg-baixa">Baixa</label>

        <input type="radio" id="urg-alta" name="urgencia" value="alta">
        <label for="urg-alta">Alta</label>

        <!-- Área de Texto Múltiplas Linhas -->
        <label for="desc">Descrição:</label>
        <textarea id="desc" name="descricao" rows="4"></textarea>

        <!-- Botão de Submissão -->
        <button type="submit">Enviar Chamado</button>

    </fieldset>
</form>
