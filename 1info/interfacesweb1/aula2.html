Pintando as Paredes da Caixa

Se nas aulas anteriores nós levantamos as paredes das caixas e organizamos o tráfego delas, agora chegou o momento de dar acabamento. Adicionar fundos e transparências altera diretamente a percepção de profundidade do utilizador.

Gradientes e Imagens

O gradiente (`linear-gradient`) funde duas ou mais cores simulando o efeito de luz natural na tela. Já as imagens de fundo cobrem a estrutura, precisando de regras de tamanho para não se repetirem de forma desordenada.

Opacidade e RGBA

A opacidade cria o efeito de vidro. O comando `opacity` deixa a caixa inteira invisível (incluindo as letras), enquanto o canal Alfa do `rgba()` permite deixar apenas a tinta de fundo transparente, poupando o texto.

Regra de Ouro da Estética:
Cuidado com o contraste! Um fundo bonito com muitas cores pode destruir a leitura das letras se você não escolher a cor do texto com sabedoria.

1. Ferramentas de Estilização

Fundo de Imagem

A propriedade background-image: url('...'); joga uma fotografia para dentro do fundo da caixa, cobrindo a cor sólida padrão.

Controle de Repetição

O comando background-repeat: no-repeat; impede que o navegador faça um efeito de "mosaico", forçando a imagem a aparecer uma única vez.

Cobertura Total

Usar background-size: cover; ordena que a imagem estique ou encolha para cobrir 100% do espaço disponível da parede da caixa.

Filtro de Opacidade

A propriedade opacity: 0.5; define o nível de transparência numa escala de 0 (totalmente invisível) a 1 (totalmente sólido).

2. Prática: Aplicando Efeitos Visuais

Veja no exemplo abaixo como criamos uma página com fundo degradê moderno e um cartão de conteúdo semi-transparente usando arquivos isolados:

<!-- =========================================================
     ARQUIVO: index.html
     ========================================================= -->
<!DOCTYPE html>
<html lang="pt-br">

    <meta charset="UTF-8">
    <title>Efeitos Visuais de Fundo</title>
    <link rel="stylesheet" href="estilo.css">

<body>

    <div class="box-principal">
        <h2>Efeito de Transparência</h2>
        <p>Note como o fundo da página consegue ser visto levemente através desta caixa branca.</p>
        <div class="alerta-opaco">Aviso: Caixa com opacidade total ativa!</div>
    </div>

</body>
</html>


/* =========================================================
   ARQUIVO SEPARADO: estilo.css
   ========================================================= */

body {
    margin: 0;
    padding: 50px;
    font-family: sans-serif;
    
    /* GRADIENTE: Cria uma fusão que começa no roxo e termina no azul para a direita */
    background-image: linear-gradient(to right, #6a11cb, #2575fc);
    background-size: cover;
    background-repeat: no-repeat;
    height: 100vh;
}

.box-principal {
    max-width: 500px;
    margin: 0 auto;
    padding: 30px;
    border-radius: 15px;
    color: #333333;
    
    /* CANAL ALFA (RGBA): Fundo branco Puro (255,255,255) com 85% de força. 
       Apenas o FUNDO fica transparente. O texto acima continua 100% nítido. */
    background-color: rgba(255, 255, 255, 0.85);
}

.alerta-opaco {
    background-color: #e74c3c;
    color: white;
    padding: 15px;
    margin-top: 20px;
    border-radius: 8px;
    
    /* OPACIDADE TOTAL: Deixa o componente inteiro fantasmagórico (fundo e texto juntos) */
    opacity: 0.4;
}
