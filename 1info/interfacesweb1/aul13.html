# Visão Geral

Nesta aula prática, vamos elevar o nível visual das nossas interfaces. Entenderemos como as propriedades box-shadow, border-radius, overflow e transition trabalham juntas para simular profundidade e interações orgânicas no ecossistema web.

1. Conceitos Chave

box-shadow

Gera sombras projetadas. Cria a ilusão de que o elemento está flutuando sobre a página (eixo Z).

border-radius

Suaviza as quinas do Box Model. Essencial para o design moderno e interfaces amigáveis.

overflow: hidden

Funciona como uma guilhotina. Impede que filhos (como imagens) vazem pelos cantos arredondados do pai.

transition

Interpola estados. Transforma mudanças abruptas de CSS (como o hover) em animações fluidas.

2. Código do Projeto Base

Copie os blocos abaixo para reproduzir o projeto no seu editor local.

index.html HTML5 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Card Moderno</title>
    <link rel="stylesheet" href="estilo.css">
</head>
<body>

    <div class="card">
        <img src="https://images.unsplash.com/photo-1603302576837-37561b2e2302?auto=format&fit=crop&w=600&q=80" alt="Notebook Gamer">
        <h2>Notebook Gamer</h2>
        <p>Intel Core i7 • 16GB RAM</p>
        <button>Comprar</button>
    </div>

</body>
</html>
estilo.css CSS3 
body {
    background: #f1f5f9;
    font-family: sans-serif;
    padding: 40px;
}

.card {
    max-width: 350px;
    margin: 0 auto;
    background: white;
    border-radius: 18px;
    overflow: hidden; /* Corta a imagem na borda */
    box-shadow: 0 10px 25px rgba(0,0,0,.20);
    transition: .3s;
}

.card:hover {
    transform: translateY(-8px);
    box-shadow: 0 20px 40px rgba(0,0,0,.30);
}

img {
    width: 100%;
    display: block;
}

h2, p {
    padding: 0 20px;
}

button {
    width: 100%;
    padding: 15px;
    background: #2563eb;
    color: white;
    border: none;
    cursor: pointer;
}
